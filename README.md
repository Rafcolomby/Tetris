# Tetris em C

Projeto desenvolvido em linguagem C com implementação da lógica do jogo Tetris.

## Objetivo
Aplicar conceitos de lógica de programação, estruturas condicionais e laços de repetição por meio da construção de um jogo interativo no terminal.

## Funcionalidades implementadas
- Controle das peças
- Movimentação e rotação
- Verificação de colisão
- Atualização da pontuação
- Lógica de finalização do jogo

## Conceitos aplicados
- Estruturas condicionais (if / else)
- Laços de repetição
- Manipulação de variáveis
- Organização da lógica do jogo
- Controle de fluxo

## Status
Projeto acadêmico concluído.
